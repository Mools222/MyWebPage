Features of this website

Click these list items to see more info about the features of this website.

  • Flexbox

  • This page is built using Flexbox.

    Learn more about Flexbox here:
    https://www.w3schools.com/css/css3_flexbox.asp

  • Sticky Navigation Bar

  • Scroll down and see that the top navigation bar is sticky.

    Learn more about navigation bars here:
    https://www.w3schools.com/css/css_navbar.asp

  • Responsive Web Design (RWD)

  • By using media queries the style of this website adapts to the size of the user's browser. See the effect by making the browser less wide.

    Learn more about RWD here:
    https://www.w3schools.com/html/html_responsive.asp

  • Unordered List (<ul>)

  • This is an unordered list changed via CSS.

    Learn more about unordered lists and styling them here:
    https://www.w3schools.com/html/html_lists.asp
    https://www.w3schools.com/css/css_list.asp

  • JavaScript

  • Opening and closing these list items works via JavaScript. So does the button on the CompSci page. Go click it.

    Learn more about JavaScript here:
    https://www.w3schools.com/js/default.asp

  • Iframe

  • Scroll down to the bottom of the About Me page to see an Iframe.

    Learn more about Iframes here:
    https://www.w3schools.com/html/html_iframe.asp

  • Search / HTML Form / PHP

  • The search bar is a HTML form. The form-handler uses PHP. Try searching for something. Not much happens, but it's something.

    Learn more about forms here:
    https://www.w3schools.com/html/html_forms.asp
